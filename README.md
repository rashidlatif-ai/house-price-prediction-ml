# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using the Ames Housing dataset. It is an end-to-end machine learning regression project that includes data preprocessing, feature engineering, model training, and evaluation.

---

## Problem Statement
The goal is to predict the final sale price of residential homes based on various features such as location, size, quality, and construction details.

---

## Dataset
- Source: Ames Housing Dataset
- Training samples: 1460 rows
- Features: 79 explanatory variables
- Target variable: SalePrice

---

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

---

## Workflow

1. Data Cleaning & Missing Value Handling  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering & Encoding  
4. Model Training:
   - Linear Regression (Baseline)
   - Random Forest
   - XGBoost (Final Model)
5. Model Evaluation using RMSE  
6. Final Prediction on Test Data  

---

## Results
- Best Model: XGBoost Regressor  
- Performance metric: RMSE (Root Mean Squared Error)

---

## Output
Final predictions were generated and saved in `submission.csv`.

---

## Key Learnings
- Handling real-world messy datasets  
- Feature engineering for regression problems  
- Comparing multiple ML models  
- Improving performance using XGBoost
